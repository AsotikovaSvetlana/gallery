### Технологии

---

### Быстрый старт
- Клонируйте репозиторий: `git clone https://github.com/AsotikovaSvetlana/gallery.git`
- Перейдите в папку с проектом: `cd gallery`
- Установите зависимости: `npm install`
- Запустите проект: `npm start`
